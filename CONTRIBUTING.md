## Настройка разработки

1. Сделайте форк репозитория
2. Клонируйте свой форк: `git clone https://github.com/your-username/reindexer-langchain-community.git`
3. Создайте виртуальное окружение: `python -m venv venv`
4. Активируйте виртуальное окружение: `source venv/bin/activate` (Linux/Mac) или `venv\Scripts\activate` (Windows)
5. Установите пакет в режиме разработки: `pip install -e .[dev]`
6. Запустите тесты: `python -m pytest tests/`

## Руководство по стилю

Этот проект использует ruff для форматирования кода и проверки на ошибки. Пожалуйста, убедитесь, что ваш код следует этим рекомендациям:

* Используйте 88 символов в качестве максимальной длины строки
* Следуйте руководству по стилю PEP 8 для кода на Python
* Пишите строки документации для всех публичных методов и классов
* Используйте аннотации типов, где это возможно

Для форматирования кода запустите: `ruff format .`

Для проверки на ошибки: `ruff check .`

## Тестирование
